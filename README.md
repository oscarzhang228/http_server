# HTTP Server Built in Python

Built to understand the HTTP protocol better.

## Features

### Web Server

**Supported Versions**

- [x] HTTP/1.1
  - Following [RFC 2616, Section 5](https://datatracker.ietf.org/doc/html/rfc2616#section-5)
- [ ] HTTP/2
- [ ] HTTP/3

**Supported Methods**

- [ ] OPTIONS
- [ ] GET
- [ ] HEAD
- [ ] POST
- [ ] PUT
- [ ] DELETE
- [ ] TRACE

#### Request Parsing

**Parts**

- [x] Request-Line
- [ ] General-Headers
- [ ] Request-Headers
- [ ] Entity-Headers
- [ ] Query Params
